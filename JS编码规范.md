### 1. 缩进
使用 4 个空格做为一个缩进层级，不允许使用 2 个空格 或 tab 字符。

```javascript
	// 不好的写法
	if (a && b) {	
      for (i = 0, i < 10; i++) {	
        alert(i);	
      }	
    }
	// 好的写法  
    if (a && b) {	
        for (i = 0, i < 10; i++) {	
            alert(i);	
        }	
    }	
```
